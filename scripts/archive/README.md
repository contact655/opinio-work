# scripts/archive

このディレクトリは、開発中に作成されたが現時点で使用しないスクリプトを保管します。

## seed-opinion-data.sql

### 概要
- Phase γ-1 (2026-05-02) で作成された、ow_companies の fit_positives / fit_negatives サンプル投入スクリプト
- 30 社分のテキストが業種・フェーズ固有の観察として書かれている（テンプレ感なし）

### なぜ archive にあるか
- WHERE 句が `name = 'テスト株式会社_XX'` で書かれているが、本番 DB の name はすべて「テスト株式会社_」で番号なし
- そのまま実行しても 0 行更新で空振りする
- Phase γ-1 の動作確認は scripts/seed-opinion-sample-3.sql で 3 社分のみ投入する形で完結した

### 将来の再利用方法
1. Phase ε (Supabase MCP 接続) 完了後、MCP で本番 DB の各企業の UUID を取得
2. 各企業の業種・フェーズ情報を取得して、このスクリプトの「業種・フェーズコメント」とマッピング
3. WHERE 句を `WHERE id = '<UUID>'` に書き換える（ヒアドキュメント化やスクリプト生成も可）
4. 30 社分のテキスト品質を活かしたまま再投入可能

### 注意事項
- このスクリプトのテキストはサンプルであり、本番運用では編集部が取材して書いた見立てに差し替える前提
- 再投入時にも「最終的には編集部の手による書き直しが必要」という認識は維持すること
